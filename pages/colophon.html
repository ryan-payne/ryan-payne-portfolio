The Tech Stuff

This site was written in plain old HTML and SCSS (like we used to do it), was built with Jekyll, a Ruby static site generator, and is hosted on Github Pages.

This code is certified CSS framework free.

The Design Stuff

This site was typeset in Inter by Rasmus Andersson.

Inter is a typeface carefully crafted & designed for computer screens. It features a tall x-height to aid in readability of mixed-case and lower-case text.
